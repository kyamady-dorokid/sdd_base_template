# Claude(.claude/skills) と Codex(.agents/skills) の「構造パリティ」検証用の許容リスト。
#
# cc-sdd は Claude(/kiro 記法) と Codex($kiro 記法・agents/ サブディレクトリ) で
# スキルの *内容* を意図的に変えて生成する。したがって内容のバイト一致は検証しない。
# 検証するのは「両エージェントに同じスキル群（ディレクトリ名の集合）が存在するか」。
#
# このファイルには「片側のエージェントにのみ存在してよいスキル名」を1行1つ列挙する。
# 通常は空（両者に同一スキルが揃うのが正）。想定外の片側欠落は検証で停止・報告される。
#
# 例:
# some-claude-only-skill
